class="container-hybrid"><h1 class="title">nix guide</h1></div></header><div class="container-hybrid"><div class="docs-guide row"><div class="docs-content col-xs-16 col-md-9"><div class="guide-content"><h6>This library was created to help you customize the structure of your project with classes that change properties such as margins and alignments, including support for responsive.</h6><article id="1"><h2>Getting started</h2><p>Download Nix from <a href="https://github.com/aurumsoftware/nix" target="_blank">repository</a> or install the package <code>aurum-nix</code>:</p><div class="code-container"><button class="btn btn-sm btn-copy"><span class="icon-12-overlap"></span></button><pre><code class="code" data-mode="shel" ref="code">yarn add aurum-nix</code></pre></div><p>and import into your project:</p><div class="code-container"><button class="btn btn-sm btn-copy"><span class="icon-12-overlap"></span></button><pre><code class="code" data-mode="xml" ref="code"><link rel="stylesheet" type="text/css" href="node_modules/aurum-nix/dist/nix.min.css" /></code></pre></div></article><article id="2"><h2>Usage</h2><p>Write the classes as you would write css properties and values separating with <code>_</code> between properties and the values.</p><p>By default, the classes have a prefix <code>nix-</code> that you should apply before the class name.</p><p>You can use the responsive classes with the suffix <code>--xs</code> for screens up to 768px (mobile), <code>--sm</code> for screens up to 1024px (tablets and small screens), <code>--md</code> for screens up to 1200px (medium screens) and <code>--lg</code> for screens up to 1600px (large screens).</p></article><article id="3"><h2>Customization</h2><p>You can customize it for your own project, change the values of variables then import the Nix file <code>node_modules/aurum-nix/nix</code> in your Sass file.</p><p><br><br></p><h3>Prefix variable</h3><p>Change the prefix name to your preferred:</p><div class="code-container"><button class="btn btn-sm btn-copy"><span class="icon-12-overlap"></span></button><pre><code class="code" data-mode="sass" ref="code">$prefix: 'nix-'; // nix- is default</code></pre></div><p>In this example, your class will be <code>nix-diplay_block</code>.</p><p>or set the prefix to a blank value:</p><div class="code-container"><button class="btn btn-sm btn-copy"><span class="icon-12-overlap"></span></button><pre><code class="code" data-mode="sass" ref="code">$prefix: '';</code></pre></div><p>With this example, your class name will be <code>diplay_block</code>.</p><p><br><br></p><h3>Breakpoints variables</h3><div class="code-container"><button class="btn btn-sm btn-copy"><span class="icon-12-overlap"></span></button><pre><code class="code" data-mode="sass" ref="code">$xs: 768px; // screens up to 768px (mobile) $sm: 1024px; // screens up to 1024px (tablet/small desktops) $md: 1200px; // screens up to 1200px (medium desktops) $lg: 1600px; // screens up to 1600px (large desktops)</code></pre></div><p><br><br></p><h3>Sizes variables</h3><p>These sizes are used for margins and paddings:</p><div class="code-container"><button class="btn btn-sm btn-copy"><span class="icon-12-overlap"></span></button><pre><code class="code" data-mode="sass" ref="code">$int-size: 5; // every 5px $max-size: 100; // up to 100px</code></pre></div><p><code>$int-size</code> is the interval variable to sizes, and <code>$max-size</code> is the max size to generate.</p><p><br><br></p><h3>Generate your custom values</h3><ol><li>Clone the <a href="https://github.com/aurumsoftware/nix" target="_blank">Nix repository</a>;</li><li>Install <code>gulp-cli</code>;</li><li>Run <code>yarn</code> or <code>npm instal</code>;</li><li>Run <code>gulp</code> to build or <code>gulp watch</code> to watch the changes;</li></ol></article><input type="hidden" value="nix"><input type="hidden" value="nix"></div><div
